# SIGIL — full-feature dependency set
#
# This file installs everything needed for the LLM adapters, audit proxy,
# and precise token accounting to work out of the box. For a minimal install
# (core seal/audit/runtime only), `pip install pynacl` is sufficient. For a
# fully-supported install, `pip install -e ".[all]"` is equivalent to this
# file and is the pyproject.toml-native path.
#
# See pyproject.toml for the canonical dependency declaration. The
# optional-dependencies groups defined there are:
#   [llm]    httpx + python-dotenv (LLM adapters, .env support)
#   [tokens] tiktoken (precise token counts; heuristic fallback otherwise)
#   [all]    everything in [llm] + [tokens]
#   [dev]    everything plus pytest

# Hard requirement — used by the cryptographic core (seals, signatures,
# audit chain signing, encrypted state at rest, Argon2id key wrapping).
pynacl>=1.5.0

# Required if you use sigil_audit_proxy or sigil_llm_adapter
# (Claude / OpenAI / Gemini / Ollama HTTP transport).
httpx>=0.25.0

# Optional .env file loading for adapter API key configuration.
python-dotenv>=1.0.0

# Optional precise token counting. Without this, the cost calculator
# falls back to a heuristic (still functional, less accurate).
tiktoken>=0.7.0

# ---------------------------------------------------------------------------
# Provider-specific SDKs are NOT needed; SIGIL adapters speak each provider's
# HTTP API directly via httpx. Uncomment only if you want to call providers
# outside the adapter path.
# ---------------------------------------------------------------------------
# anthropic>=0.18.0
# openai>=1.0.0

# ---------------------------------------------------------------------------
# Development dependencies (use `pip install -e ".[dev]"` instead).
# ---------------------------------------------------------------------------
